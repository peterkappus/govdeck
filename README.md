# GovDeck

Contents:
- index.html
- watch.sh
- serve.sh
- README.md

## Features
- Typography-first slides
- GOV.UK inspired styling
- DSIT identity block
- Auto-scaling headings
- Fullscreen (F)
- Fragment reveals
- title, section, content, stats, quote, image, two-column layouts

## Run
./serve.sh

Browse:
http://localhost:8000

## Update workflow
Run ./watch.sh then download new index.zip files into Downloads.
Files are copied into ~/code/govdeck.
.git is preserved.
